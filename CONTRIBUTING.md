# Mitwirken bei Boschettissimo

Vielen Dank, dass du bei Boschettissimo mitwirken möchtest! Wir freuen uns über jede Art von Beitrag, sei es das Melden von Fehlern, das Vorschlagen neuer Funktionen oder das Einreichen von Code-Änderungen.

## Fehler melden

Wenn du einen Fehler findest, öffne bitte ein **Issue** auf GitHub.

- Beschreibe den Fehler so genau wie möglich.
- Füge Schritte zur Reproduktion hinzu.
- Nenne deine Umgebung (Betriebssystem, Browser, Version).

## Funktionen vorschlagen

Hast du eine Idee, wie wir das Projekt verbessern können?

- Öffne ein Issue und beschreibe deine Idee.
- Erkläre, warum diese Funktion nützlich wäre.

## Code beisteuern

1.  **Fork** das Repository und klone es lokal.
2.  Erstelle einen neuen **Branch** für deine Änderung (`git checkout -b feature/meine-aenderung`).
3.  Nimm deine Änderungen vor und beachte dabei unseren **Coding Style** (siehe unten).
4.  Führe, wenn möglich, Tests aus, um sicherzustellen, dass nichts kaputtgegangen ist.
5.  **Committe** deine Änderungen (`git commit -m 'Füge Feature X hinzu'`).
6.  **Push** den Branch auf deinen Fork.
7.  Erstelle einen **Pull Request (PR)** gegen unseren `main` Branch.
